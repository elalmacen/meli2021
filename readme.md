# Challenge Mercado Libre 2021

Solucion:

https://dropbox.com/

## Requisitos

 - Node >= v13.10.1
 - Git

## Instalacion

    cd front/meli
    npm install
    cd ../..
    cd back 
    npm install

## Instrucciones

### Desde /back

    npm start

Endpoints:
http://localhost:3011/api/items?q=q
http://localhost:3011/api/items/:item

### Desde /front

    npm start

Entrar en http://localhost:3000
